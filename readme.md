### Text Selector

Paste in some text, make selections and see them show up on the side.

Try it out [here](http://www.sinanguclu.co.uk/text_selection/).

Depends on:

```
Bootstrap 3.3.7
```

```
jQuery 3.1.1
```

```
Mustache 2.3.0
```

#####To run server:

Install python

Clone the repo

From the root of the repo run:

```
python -m SimpleHTTPServer
```
